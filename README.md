# AI-Powered Spam & Phishing Detection System

This project builds a machine learning model to detect spam and phishing messages using natural language processing techniques.

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn

## Features
- Data preprocessing and cleaning
- Text transformation using TF-IDF Vectorizer
- Spam/phishing classification using Naïve Bayes algorithm
- Model evaluation

## Dataset
- SMS Spam Collection Dataset

## Results
- The model successfully classifies spam and non-spam messages with good accuracy

## How to Run
1. Open the notebook in Google Colab
2. Run all cells step by step
